How did you display the map in different projections using QGIS?

This assignment involved representing the same data in various different projections through QGIS. This was done by loading in the data as usual, but the projection that we view the data through was manipulated. I used the projection tab towards the bottom right of the QGIS screen to change the projection. The indicatrix plugin was used to create an overlay revealing the distortions that each different projection contributed to.
